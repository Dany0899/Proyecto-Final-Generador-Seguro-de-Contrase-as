# Generador Seguro de Contraseñas en Python

## Proyecto Integrador

### Integrante
Daniel Macas

### Objetivo del Sistema
Desarrollar un programa en Python que permita generar contraseñas seguras de acuerdo con la longitud y los tipos de caracteres seleccionados por el usuario.

### Descripción
Este proyecto consiste en un Generador Seguro de Contraseñas desarrollado en PYthon. El sistema permite crear contraseñas personalizadas utilizando letras mayúsculas, letras minúsculas números y símbolos.

Además, el programa evalúa el nivel de seguridad de la contraseña generada y muestra recomendaciones para proteger mejor la información personal.

### Funcionalidades
- Generar contraseña seguras.
- Seleccionar la longitud de la contraseña.
- Elegir si se desea incluir mayúsculas, minúsculas, números y símbolos.
- Mostrar el nivel de seguridad de la contraseña.
- Presentar recomendaciones de seguridad.
- Salir del sistema medinate un menú interactivo.
